El Desafío

Integrando Datos para la Supervivencia Planetaria

El descubrimiento de objetos cercanos a la Tierra (NEOs), como el hipotético "Impactor-2025," subraya el riesgo de colisiones celestes con nuestro planeta, las cuales pueden causar devastación, incluyendo tsunamis y eventos sísmicos. Nuestra herramienta rompe el aislamiento de datos al integrar los parámetros orbitales de asteroides (tamaño, velocidad, órbita) de la API NEO de la NASA con conjuntos de datos geológicos y ambientales del USGS (como la topografía y la actividad sísmica).

Funcionalidades Clave

Una plataforma integral para el análisis, predicción y mitigación de amenazas NEO.

Simulación 3D de Trayectorias

Modelado de la órbita del asteroide utilizando elementos Keplerianos (ej., semieje mayor y excentricidad) y cálculos de posición orbital para una visualización precisa y dinámica.

Predicción de Consecuencias

Cálculo de la energía cinética del impacto y estimación de variables críticas como el tamaño del cráter o la magnitud sísmica equivalente para evaluar el nivel de amenaza.

Visualización de Riesgos Ambientales

Mapeo de efectos secundarios, como la inundación por tsunamis, empleando datos de elevación del USGS (DEMs) para modelar el impacto en ecosistemas y poblaciones.

Evaluación de Mitigación

Permite a los usuarios explorar y visualizar los resultados de estrategias proactivas de desviación, como ajustar los parámetros de un impactador cinético y observar los cambios en la trayectoria.

"Meteor Madness" es una plataforma conceptual de simulación desarrollada para demostrar la viabilidad de integrar múltiples fuentes de datos para la gestión del riesgo de asteroides. Este proyecto busca transformar la información bruta en una herramienta educativa y de apoyo a la decisión.

Fuentes de Datos

Este proyecto utiliza datos en tiempo real de:

  • NASA Near-Earth Object (NEO) Web Service API
  • USGS National Earthquake Information Center (NEIC) Earthquake Catalog (ComCat)
  • USGS National Map Elevation Data
