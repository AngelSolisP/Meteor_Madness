The Challenge

Integrating Data for Planetary Survival

The discovery of Near-Earth Objects (NEOs), like the hypothetical "Impactor-2025," highlights the risk of celestial collisions with our planet, which can cause devastation, including tsunamis and seismic events. Our tool breaks down data silos by integrating asteroid orbital parameters (size, velocity, orbit) from NASA's NEO API with geological and environmental datasets from the USGS (such as topography and seismic activity).

Key Features

A comprehensive platform for the analysis, prediction, and mitigation of NEO threats.

3D Trajectory Simulation

Modeling of the asteroid's orbit using Keplerian elements (e.g., semi-major axis and eccentricity) and orbital position calculations for a precise and dynamic visualization.

Consequence Prediction

Calculation of the impact's kinetic energy and estimation of critical variables such as crater size or equivalent seismic magnitude to assess the threat level.

Environmental Risk Visualization

Mapping of secondary effects, such as tsunami flooding, using USGS elevation data (DEMs) to model the impact on ecosystems and populations.

Mitigation Assessment

Allows users to explore and visualize the results of proactive deflection strategies, such as adjusting the parameters of a kinetic impactor and observing the changes in trajectory.

"Meteor Madness" is a conceptual simulation platform developed to demonstrate the feasibility of integrating multiple data sources for asteroid risk management. This project aims to transform raw information into an educational and decision-support tool.

Data Sources

This project uses real-time data from:

  • NASA Near-Earth Object (NEO) Web Service API
  • USGS National Earthquake Information Center (NEIC) Earthquake Catalog (ComCat)
  • USGS National Map Elevation Data
